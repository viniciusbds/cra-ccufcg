## ex:

    p1 foi pago com 9 em 60 hrs semestrais
    S1 = {"p1": (9, 60)...}


    didatica1 foi paga com 10, mas totalizando 30 hrs semestrais
    S1 = {"didatica1": (10, 30)...}
